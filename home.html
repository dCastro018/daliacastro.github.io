Wildcat Dam

Wildcat Dam is a research project sponsored by MITRE via the INSuRE program, focused on advancing cybersecurity testing for Industrial Control Systems (ICS). The project simulates a dam environment using Python and the Modbus TCP protocol, enabling control of sluice gates based on real-time water level and surge inputs. Integrated with MITRE Caldera for OT, the virtual setup replicates PLC behavior and includes process logic, sensor emulation, and visualization tools.

NeoCath App

For my senior capstone, I developed a mobile application that tracks the values of two temperature sensors and a pH sensor embedded in a smart urinary catheter. The application features reading tracking, data filtering, and alerts for healthcare providers when sensor values fall outside the normal range. Additionally, it allows providers to remotely toggle a generated electromagnetic field. The system follows a layered architecture: the backend is built with a Python Flask server, while the frontend is developed using React Native. Sensor data is transmitted from an Arduino to the backend server, where it is processed, displayed on the frontend, and stored in an AWS-hosted database.

Tickect Printing System

I developed a desktop application using Java and JavaFX for a tortilla factory to streamline the customer waiting process. The system prints sequential tickets for clients placing orders, allowing them to track their turn while waiting. It features an intuitive user interface for managing ticket numbers and integrates with thermal receipt printers using ESC/POS commands for direct communication and custom formatting. Built with Maven and tested using JUnit, the application ensured reliable performance in a fast-paced, customer-facing environment.

Scholarship and Application Management System

This is a full-stack Scholarship Management System to streamline the process of matching students with eligible scholarships. The application uses a PostgreSQL database, a Node.js and Express backend, and a React frontend. It supports detailed scholarship criteria such as GPA, major, year, and ethnicity, allowing students to view ranked matches and apply directly through the platform. The system includes tools for reviewers to score applications and for administrators to manage users and data. I implemented a GitLab CI/CD pipeline to automate testing, building, and deployment, ensuring consistent and efficient development workflows. Jest was used to create unit and integration tests, improving reliability and maintainability across both frontend and backend components.
